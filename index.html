Durdle.
Discord Wordle Bot

New Words Everyday

A new word is fetched from our word list which contains over 5700 words, and each user gets a different word to guess everyday.

Daily Streaks

each user's maximum durdle streak is stored in our database along with the number of times they played durdle. you can compare your streaks and accuracy with your friends!

Provides Meaning For The Words

durdle also provides the meaning and usage of the words which are fetched for the user, if the word is found on Word API
